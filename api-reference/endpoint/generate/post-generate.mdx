## Generation Status Updates

There are two ways to monitor 

<AccordionGroup>
  <Accordion title="Polling for completion">
    Use [GET Generation](/api-reference/endpoint/generate/get-generate) endpoint to poll for a generation's status
  </Accordion>
  <Accordion title="Using webhooks">
    Set `webhookUrl` field in the request body to recieve notifications for status updates of the generation

    To learn more about webhooks, check out how to use the [Webhooks](/api-reference/webhooks).
  </Accordion>
</AccordionGroup>
